# Paper Trading (Swisstony Maker-Only)

## 目的
在不消耗大量 Odds API 的情况下，验证 **低价挂单是否能接到恐慌盘**，并记录“成交 → 回弹”的概率。

## 核心策略
- **Maker-only**：只挂单不追价
- **低价阶梯**：目标价 = 公允价 * 0.85
- **极低频轮询**：10 分钟间隔（节省 API）

## 配置文件
- `data/watchlist.json`
  - **手动版**：直接写 `fair_prob`
  - **API版**：写 `odds_key`（从 The Odds API 的事件+队名映射）

示例：
```json
[
  {
    "name": "NBA Example - Lakers vs Warriors (MANUAL)",
    "condition_id": "0x...",
    "fair_prob": 0.60
  }
]
```

## 运行
```bash
python3 skills/poly_paper_vulture.py
```

## 日志输出
- 总表：`logs/paper_trades.csv`
- 单笔成交：`logs/fills/fill_YYYYMMDD_HHMMSS.json`
- **每笔成交都会上传 CDN + 同步 Notion**（类型=项目文档，标签=ClawBot产出）

## 注意
- The Odds API 免费版不含 Historical Odds，本脚本只使用实时赔率。
- 若 watchlist 为空或 condition_id 未填，会提示并跳过。
